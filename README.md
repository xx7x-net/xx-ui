> A Vue.js 2.0 UI Toolkit for Web.

XXUI will stay with Vue 2.x 

## Install
```shell
npm install xxqx-ui -S
```

## Quick Start
``` javascript
import Vue from 'vue'
import XXUI from 'xxqx-ui'

Vue.use(XXUI)

// or
import {
  Select,
  Button
  // ...
} from 'xxqx-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```
